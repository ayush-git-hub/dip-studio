# DIP Studio — Gonzalez & Woods Edition

**Digital Image Processing Educational Editor**  
Python · PySide6 · OpenCV · NumPy

DIP Studio is a desktop-based educational image processing software inspired by the concepts from **Gonzalez & Woods - Digital Image Processing**. It provides an interactive environment to learn and apply core image processing techniques such as filtering, enhancement, transformations, histograms, morphology, and segmentation. :contentReference[oaicite:0]{index=0}

---

## Features

- Modern desktop GUI using **PySide6**
- Image enhancement tools
- Spatial filtering & convolution kernels
- Frequency domain processing (FFT)
- Histogram visualization
- Morphological operations
- Edge detection & segmentation
- Educational chapter-based reference system

---

## Requirements

- Python 3.10+
- PySide6
- OpenCV
- NumPy
- Matplotlib

---

## Installation

### 1. Clone Project

```bash
git clone https://github.com/ayush-git-hub/dip-studio
cd dip-studio
````

### 2. Create Virtual Environment

```bash
python -m venv venv
source venv/bin/activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Run Project

```bash
python main.py
```
